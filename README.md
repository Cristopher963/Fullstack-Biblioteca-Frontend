# 📚 Biblioteca Fullstack

Aplicación web fullstack para la **gestión de una biblioteca**, desarrollada con arquitectura separada de **frontend + backend**, enfocada en la administración de libros, lectores, ejemplares y préstamos.

Este proyecto fue construido como práctica real de desarrollo fullstack, aplicando buenas prácticas en:

* arquitectura por capas
* consumo de API REST
* diseño responsive
* separación frontend/backend
* DTOs y mappers
* validaciones
* flujo CRUD completo

## ⚡ Instalación local

### 1) Clonar frontend

```bash
git clone https://github.com/Cristopher963/Fullstack-Biblioteca-Frontend
```

### 2) Clonar backend

```bash
git clone https://github.com/Cristopher963/Fullstack-Biblioteca-Backend
```

### 3) Ejecutar backend

```bash
mvn spring-boot:run
```

### 4) Ejecutar frontend

```bash
npm install
npm run dev
```
